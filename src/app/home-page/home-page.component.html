Welcome to OpenTournament!

Are you a Player? No Problem just create your Player-Profile

In OpenTournament you can register for Tournaments. The Organizer can use your registration to add you easily to the tournament you just registered for.

OpenTournament BETA For the Players!

OpenTournament is a platform to manage your tournaments. For now there is only one game supported: WM/HO. The usage of the platform is total free and there are no adds. Here is a list of key features for players:

  • Under "MySite" you see all your Tournaments/Games/Registrations
  • If you create an account and add information about your Team/Meta/Origin/Country the tournament organizer can easily prevent pairings for same team. No more driving through the half world to play against your local dude in the first round!
  • Each registered Player has an ELO number. After a tournament is finally uploaded for all player with an ELO number there is a calculation for a new ELO number. For more Information see Wikipedia.
  • A tournament is published in "RealTime". So you can see as a player all rankings and pairings immediately.
  • If you register for a tournament you can see "Your Game" marked with a yellow border.
  • You can enter the result of your game if the game is untouched (you opponent or the organizer hasn't entered something). The organizer may check (and correct) the result.
  • You can upload your ArmyList with your registration. Only the tournament will see it for now.
  • There is a option to enter your played list (you need to insert one or more ArmyList(s) in registration). After the tournament you can see the exact result of all your games. On top you can see ELO changes for each game you played.
OpenTournament BETA For the Tournament Organizers

Here is a list of same key features for Tournament Organizer:

  • Under "MySite/MyTournaments" you see all your Organized Tournaments
  • You can easily add registered players with one click. If they add a ArmyList you can check it there. No more mailing chaos!
  • Adding "offline(not registered)" players is fast and easy. Basically you only need a valid PlayerName. All other information is optional.
  • There is a check to prevent starting a tournament with uneven players. Then player 'DUMMY' comes to play.
  • You can choose to start pairing with some Restrictions. Team/Meta/Origin/Country. Each Round can be paired with separate options. You can always
  • Players can add there result on there own (check is that the game is untouched and they are one of the player). Time saving ahead!
  • Swapping Players is easy. Just drag and drop the Game Cards(Long Click). There is a check to prevent you new game parings where players already played against each other
  • Presentation modes are available. Games as a simple small table and a fullscreen/projector mode.
  • After pairing a game you can check the result of the pairings before a player can see the new round. You have to "publish Round" that all players see the round. Maybe this feature change later.
  • You can always change Game Results afterwards!! All rankings will be changed immediately.
  • Before you "Upload Tournament" you can change everything.
